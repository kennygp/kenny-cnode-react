快速实现本地dev模式运行我的源码，三步走起：
--------------
      1、克隆我的仓库；
      2、命令行 npm install ；
      3、命令行 npm run dev ，浏览器访问 localhost:8888 ，F12进入移动开发模式查看 ；
      *、如果出现问题，尝试cnpm淘宝镜像安装。
